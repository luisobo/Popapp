Popapp
=====
An easy to use and highly customizable modal popup for the web.

Basic Usage
-----
Create a new Popapp:
<pre><code>popapp = new Popapp</code></pre>
Set the contents:
<pre><code>popapp.content '&lt;div>Hello! I'm a Popupp!&lt;/div>'</code></pre>
Show the popap:
<pre><code>popapp.show()</code></pre>
